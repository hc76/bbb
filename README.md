# 疑难工单管理小程序

## 项目概述

这是一个基于微信小程序 + 云开发的疑难工单管理系统。用于管理装维团队在宽带装机过程中遇到的疑难问题，提供工单上报、流转、处理和跟踪的完整解决方案。

## 核心功能

### 1. 工单管理
- **工单上报**: 直销队员快速上报疑难工单，包含宽带账号、业务类型、下单地址等详细信息
- **工单列表**: 根据用户角色展示相应的工单
- **工单详情**: 查看完整工单信息和处理历史
- **工单回复**: 技术经理对工单进行处理和反馈
- **工单状态流转**: 待处理 → 处理中 → 已解决

### 2. 角色权限体系
- **超级管理员**: 系统维护和管理员设置权限
- **网格长/副网格长**: 监督所属网格的工单处理进度
- **技术经理**: 处理并回复多个网格的疑难工单
- **直销队员**: 上报工单并跟踪处理进度

### 3. 通知系统
- 工单创建时通知相关管理员
- 工单有新回复时通知上报人
- 工单状态变更时实时通知

### 4. 跨网格支持
- 支持跨网格工单上报
- 自动分配给对应网格的管理员
- 上报人可追踪所有相关工单

## 快速开始

### 前置要求
- 微信开发者工具 (v1.05.2105260 或更新)
- 微信小程序账号
- 云开发环境已激活

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/hc76/bbb.git
cd bbb
```

2. **在微信开发者工具中打开项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 输入项目路径和微信小程序 AppID

3. **配置云开发环境**
   - 在 `miniprogram/app.js` 中配置云开发环境 ID
   ```javascript
   wx.cloud.init({
     env: 'your-cloud-env-id'  // 替换为你的环境ID
   })
   ```

4. **部署云函数**
   - 在微信开发者工具中右键 `cloud/functions` 目录
   - 选择"上传并部署: 云端安装依赖"
   - 部署以下云函数:
     - createTicket
     - updateTicket
     - getTickets
     - getTicketDetail
     - setAdmin

5. **启动小程序**
   - 点击"编译"按钮开始开发

## 使用流程

### 直销队员上报工单
1. 打开小程序进入首页
2. 点击"上报疑难工单"按钮
3. 填写工单表单（宽带账号、下单地址、问题描述等）
4. 选择上报网格和实际网格
5. 点击"提交"按钮
6. 系统自动分配给相应网格的管理员

### 技术经理处理工单
1. 进入"工单列表"查看待处理工单
2. 点击工单查看详情
3. 在页面底部查看处理历史
4. 填写处理内容和附件
5. 选择工单状态（处理中/已解决）
6. 提交回复，上报人收到通知

### 网格长监督工单
1. 进入"工单列表"
2. 查看所属网格的所有上报工单
3. 可查看工单详情和处理进度
4. 可导出工单报表

## 文档说明

### 核心文档
- **docs/DATABASE_SCHEMA.md**: 详细的数据库集合结构设计
- **docs/API_DOCS.md**: 所有云函数的接口说明和参数定义
- **docs/USER_GUIDE.md**: 各角色的使用指南和工作流程

### 数据库设计
系统使用以下主要集合:
- **users**: 用户信息和权限管理
- **grids**: 网格信息和网格管理员
- **tickets**: 工单主表
- **messages**: 消息通知表

### API接口
主要的云函数接口:
- `createTicket`: 创建工单
- `updateTicket`: 回复和更新工单
- `getTickets`: 获取工单列表（权限过滤）
- `getTicketDetail`: 获取工单详情
- `setAdmin`: 设置管理员（仅超级管理员）

## 权限矩阵

| 操作 | 超级管理员 | 网格长 | 副网格长 | 技术经理 | 直销队员 |
|------|----------|--------|---------|---------|----------|
| 创建工单 | ✓ | ✗ | ✗ | ✗ | ✓ |
| 查看工单 | ✓ (全部) | ✓* | ✓* | ✓** | ✓*** |
| 回复工单 | ✓ | ✗ | ✗ | ✓ | ✗ |
| 设置管理员 | ✓ | ✗ | ✗ | ✗ | ✗ |
| 归档工单 | ✓ | ✗ | ✗ | ✓ | ✗ |

**注**: 
- `*` 仅能查看所属网格的工单
- `**` 能查看所管理的多个网格的工单
- `***` 仅能查看自己上报的工单

## 安全性

- **权限验证**: 每个云函数调用都会验证用户权限
- **数据隔离**: 用户只能访问有权限的数据
- **审计日志**: 重要操作都会被记录
- **加密传输**: 所有数据传输都使用 HTTPS

## 故障排除

### 工单无法创建
- 检查云开发环境是否已激活
- 确认云函数已正确部署
- 检查数据库集合权限设置

### 无法接收通知
- 检查消息集合是否正确配置
- 确认用户已授予消息通知权限

### 权限问题
- 确认用户角色设置正确
- 检查网格分配是否正确
- 验证用户ID是否匹配
