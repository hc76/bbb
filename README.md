# 疑难工单管理小程序

## 项目概述

这是一个基于微信小程序 + 云开发的疑难工单信息收集及处理反馈系统。用于管理装维师傅在宽带装机过程中遇到的疑难情况。

## 系统架构

- **前端**: 微信小程序（WXML + WXSS + JavaScript）
- **后端**: 云开发（云函数 + 数据库）
- **数据库**: Cloud Firestore 集合
- **存储**: 云存储（用于上传附件）

## 核心功能

### 1. 角色管理
- **超级管理员**: 设置各网格的管理员
- **网格管理员**: 包括网格长、副网格长、技术经理
- **直销队员**: 上报疑难工单

### 2. 工单管理
- 工单上报（宽带账号、业务、下单日期、下单地址、施工班组、客户诉求、装机疑难问题等）
- 工单状态流转（未处理 → 处理中 → 已解决）
- 工单分发（按网格自动分发给相应管理员）
- 跨网格工单追踪（上报人可查看自己上报但归属其他网格的工单）

### 3. 权限体系
- 网格长/副网格长: 查看自己网格的所有工单
- 技术经理: 可查看并回复自己负责的多个网格的工单
- 直销队员: 查看自己上报的工单及其进度
